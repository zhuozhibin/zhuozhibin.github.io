2018-01-10
NodeTutorial
Javascript, Node

1. 概述

模块能被用来组织大型的项目以及分发Node项目。Node应用由模块组成，采用CommonJS规范。

2. 模块引用

通过require方法引用模块到当前上下文。

3. 模块定义

模块内定义的对象和方法默认都是模块私有的，必须导出才能给外部引用。导出需要用到module对象。
